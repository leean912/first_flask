# Create a folder and cd into your folder

# Flask Installation Windows
```
1. py -3 -m venv .venv
2. .venv\Scripts\activate
3. pip install Flask
```

# Flask Installation MacOS/Linux
```
1. python3 -m venv .venv
2. .venv/bin/activate
3. pip install Flask
```

# Run your flask app
```
flask --app project_name run
```
